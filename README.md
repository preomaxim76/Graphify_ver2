All about operators:

1. Usual operators (operators)
x * -- multiplication: 2 * x
x / -- division: x / 2, division by 0 is impossible
x + -- plus: x + 2
x - -- minus: x -2
x ^ -- power of: x^2, x^(-1)

2. Pro Operators (pro_operators)
x log -- logarithm, log10[x] - log of x to the power of 10, log[x] - log of x to the power of 2
x sqrt -- root, sqrt[x] - square root of x, sqrt10[x] - 10 root of x
x sin/cos/tg/ctg -- sin/cos/tg/ctg, sin[30] = 1/2, sin2[30] = (1/2)^2 = 1/4

3. Brackets
() - opening and closing bracket

4. Variables (var)
Any letter from a-z (lowercase), except for y
Can be used only one variable (although can be repeated many times):
y = x^2 + b + c - wouldn't pass 
y = x^2 + x - would pass

Rules:

Every function should start with 'f(x)' or 'y';
Every function should have only one '=' sign;
Function cannot be blank;
Fast multiplication (5x) can only be used with variables(x, z, etc), brackets() or pro_operators (sin, cos, sqrt, etc);
Variable must be of length 1 and not y;
